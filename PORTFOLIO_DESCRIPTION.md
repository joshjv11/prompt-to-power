# PromptBI - Portfolio Description

## Short Description (1-2 sentences)
**PromptBI** is an AI-powered web application that transforms natural language descriptions into production-ready Power BI dashboard specifications. Users describe what they want in plain English, and the app generates complete dashboard specs with DAX measures, M code, and visualization configurations in seconds.

---

## Medium Description (1 paragraph)

**PromptBI** is an innovative AI-powered dashboard generator that bridges the gap between business users and complex data visualization tools. Built with React, TypeScript, and Google Gemini AI, the application allows users to upload CSV/Excel files and describe their dashboard requirements in natural language. The AI then generates complete Power BI dashboard specifications including DAX measures, M code, and visualization configurations—eliminating the need for technical expertise or Power BI Desktop licenses during the design phase. With features like real-time data quality validation, interactive refinement chat, undo/redo functionality, and comprehensive export options, PromptBI makes professional dashboard creation accessible to everyone in under 30 seconds.

---

## Detailed Description (2-3 paragraphs)

**PromptBI** revolutionizes how businesses create data dashboards by combining the power of artificial intelligence with an intuitive natural language interface. Unlike traditional BI tools that require extensive training and technical knowledge, PromptBI allows users to simply upload their data files (CSV or Excel) and describe what they want in plain English. The application leverages Google Gemini AI to understand user intent and automatically generates complete Power BI dashboard specifications in seconds.

Built with modern web technologies including React, TypeScript, Vite, and shadcn/ui, PromptBI offers a seamless user experience with features like interactive onboarding tutorials, real-time data quality validation, and an AI-powered refinement chat that allows users to iteratively improve their dashboards through natural language conversations. The application supports 15+ chart types (bar, line, pie, table, scatter, histogram, funnel, gauge, waterfall, and more) and exports production-ready Power BI templates with complete DAX measures and M code.

The platform addresses a critical pain point in business intelligence: the time and expertise required to build effective dashboards. By reducing dashboard creation from hours of manual work to a 30-second AI-powered process, PromptBI democratizes data visualization and empowers business users to create professional dashboards without technical barriers or expensive software licenses during the design phase.

## Technical Highlights

- **AI-Powered**: Google Gemini 2.5 Flash for natural language understanding and dashboard generation
- **Modern Stack**: React 18, TypeScript, Vite, Zustand for state management
- **UI/UX**: shadcn/ui components, Tailwind CSS, Framer Motion animations
- **Features**: Data quality validation, undo/redo, keyboard shortcuts, analytics tracking
- **Export Formats**: Power BI Template (PBIT), PDF, CSV, JSON with DAX measures
- **Performance**: LRU caching, data sampling for large datasets, virtual scrolling
- **Accessibility**: Full ARIA labels, keyboard navigation, screen reader support
